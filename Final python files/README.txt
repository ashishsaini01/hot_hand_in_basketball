In this folder we have all the data and programming file, that we are created. 

Data Folder: In this folder we have all data files, that are first scrapped from the web and it also contains the pre-processed data files.

scrap_data.R: This file is used for web scrapping part and it is a R file.

Basketball_data_formation.ipnyb : This a jupyter python notebook, by using this file we preprocessed the data.

mavi_analysis.ipnyb: In this file we perform analysis for Maxi Kleber.

paul_analysis.ipnyb: In this file we perform analysis for Chris Paul.

analysis_for_giannis_doncic_curry.ipnyb: In this file we perform analysis for Giannis, Curry and Doncic.
